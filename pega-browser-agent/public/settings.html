MCP Server Connections

Connect to external MCP (Model Context Protocol) servers to extend the agent's capabilities with additional tools and resources.

Exposed MCP Tools

These tools are exposed by this extension and can be used by external MCP clients.

pega_get_case_summary tool

Get an AI-generated summary of the current Pega case

pega_execute_action_plan tool

Execute a sequence of browser automation actions on a Pega case

pega_get_dom_snapshot tool

Get a semantic snapshot of the current Pega UI

pega_detect_framework tool

Detect if the current page is a Pega application

pega_update_field tool

Update a specific field in the current Pega case

pega_click_action tool

Click an action button in the Pega UI

pega_navigate_case tool

Navigate to a specific case or work queue in Pega

pega_wait_for tool

Wait for a specific condition in the Pega UI

Exposed MCP Resources

These resources are exposed by this extension for external MCP clients to read.

pega://current-case resource

Current Pega case with all field values and metadata

pega://case-history resource

History of actions taken on the current case

pega://available-actions resource

List of actions available for the current case
